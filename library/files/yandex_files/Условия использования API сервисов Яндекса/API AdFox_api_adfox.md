 **Сервис** — сервис «APIADFOX», представляющий собой интерфейс программирования, который \[предоставляет доступ к функционалу сервиса ADFOX, предназначенный для управления размещением Материалов в сети Интернет согласно Условиям, получения информационно\-статистической информации, включая, но не ограничиваясь указанным.

 **Данные** — сведения, поступающие от Сервиса, предназначенные для показа Пользователям, включая, но не ограничиваясь, пользовательские данные, данные систем автоматизированного учета информации, принадлежащие Яндексу, доступные Партнеру через Интерфейс Сервиса, которые в том числе могут содержать (в зависимости от назначения системы учета) сведения об использовании Партнером Сервиса.
