  1\. Общие условия
-----------------

 1\.1\. Термины и определения

 **Сервис** — сервис «API Аудиторий», представляющий собой интерфейс программирования, который предлагает Пользователю возможность отправлять запросы к сервису «Аудитории» с целью использования сервиса «Аудитории».

 **API\-Ключ** — уникальный идентификатор, выдаваемый Пользователю, позволяющий получить доступ к функциям Сервиса.

 **Данные** — сведения, поступающие от Сервиса, предназначенные для показа Пользователю, состав которых предусмотрен Правилами предоставления сервиса «Аудитории», размещенными по адресу: [https://yandex.ru/legal/audience\_tos](/legal/audience_tos/index.html).

* Лицензией на использование поисковой системы Яндекса, размещенной по адресу: [https://yandex.ru/legal/termsofuse](/legal/termsofuse/index.html);
